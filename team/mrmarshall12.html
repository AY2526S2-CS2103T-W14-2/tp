Project: ServeMate

ServeMate is a desktop application created for administrative staff of tingkat catering businesses to manage their customer contacts and schedule deliveries.

Given below are my contributions to the project.

  • Code contributed: RepoSense Dashboard

  • New Features implemented:

    • reschedule command
      • What it does: Allows administrative staff to edit a delivery that is associated with the specified customer.
      • Justification: Users may need to reschedule a delivery for a specific date and time. This feature allows them to do so without having to unschedule and schedule a delivery back with updated details, which is more efficient and less error-prone.

  • Enhancements implemented:

    • Enhanced Storage Component
      • What it does: Allows the application to store Delivery as a JSON object nested within a Person object.
      • Justification: A crucial feature to ensure data persistence for delivery details associated with a customer.
